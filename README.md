# Bremer Community Spiel

## Beschreibung

Das **Bremer Community Spiel** ist ein Canvas-basiertes Spiel, das auf einer Webseite läuft und von der Bremer Community gemeinsam entwickelt werden kann. Spieler können sich durch eine virtuelle Welt bewegen, Aufgaben von NPCs erhalten und verschiedene Interaktionen erleben. Das Spiel ist so konzipiert, dass es auf einer eigenen Domain läuft und in eine WordPress-Seite eingebunden werden kann.

## Funktionen

- **Vollbild-Canvas**: Das Spiel läuft auf einem Canvas, das die gesamte Seite ausfüllt.
- **NPCs und Spieler**: NPCs sind auf dem Canvas positioniert, und der Spieler wird als rotes Rechteck dargestellt.
- **Minimap und Inventar**: Eine Minimap hilft der Orientierung und ein einfaches Inventarsystem zeigt gesammelte Gegenstände an.
- **Dialogsystem**: Aufgaben und Informationen von NPCs werden in einer Dialogbox am unteren Rand des Bildschirms angezeigt.
- **Hintergrundmusik**: Eine Hintergrundmusik wird während des Spiels abgespielt.
