# Bremer Community Spiel

## Beschreibung

Das **Bremer Community Spiel** ist ein Canvas-basiertes Spiel, das auf einer Webseite läuft und von der Community gemeinsam entwickelt werden kann. Das erste Level bietet mindestens 10 Minuten Spielzeit durch eine große, abwechslungsreiche Umgebung, NPCs mit unterschiedlichen Aufgaben und ein Inventarsystem.

## Funktionen

- **Vollbild-Canvas**: Das Spiel läuft auf einem Canvas, das die gesamte Seite ausfüllt.
- **Erweiterte Umgebung**: Ein größeres Spielfeld mit verschiedenen Bereichen wie Dorf, Wälder und Labyrinthe.
- **NPCs und Aufgaben**: Verschiedene NPCs mit spezifischen Aufgaben und Informationen.
- **Dialogsystem**: Aufgaben und Informationen von NPCs werden in einer Dialogbox am unteren Rand des Bildschirms angezeigt.
- **Inventarsystem**: Spieler können Gegenstände sammeln und im Inventar anzeigen lassen.
- **Minimap**: Eine kleine Karte zeigt den Fortschritt und wichtige Orte.
- **Hintergrundmusik**: Eine Hintergrundmusik wird während des Spiels abgespielt.
